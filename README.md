# Sample Express

## Description
Sample application made with [Typescript](https://www.typescriptlang.org/) to show a basic express server that serves an API.

## Install
```bash
yarn install
```

## Start Application
1. Build the application by running:
```bash
yarn build
```
2. Start the application by running:
```bash
yarn start
```

## Watch mode
```bash
yarn watch
```
